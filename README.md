# Integrated Healthcare Information Management System

This project is a SQL-based Healthcare Database Management System.

## Features
- Department, Doctor and Patient management
- Appointment scheduling
- Prescription tracking
- Billing system
- SQL Joins, Views, Subqueries
- Stored Procedures and Triggers
- Transactions and Aggregate Functions

## Technologies Used
- MySQL
- SQL

## Tables
- Department
- Doctor
- Patient
- Appointment
- Prescription
- Billing

This project demonstrates database design and advanced SQL concepts.
